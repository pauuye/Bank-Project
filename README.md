**Bank Analytics Project (MySQL, Excel, Power BI)**

This is a beginner-friendly analytics project using MySQL, Excel, and Power BI to explore banking data.
The project focuses on:

Customer behavior

Transactions

Payment methods

Country trends

Completion and failure rates

Monthly and yearly transaction activity

This project shows end-to-end skills:

data cleaning → EDA → dashboards.

**What This Project Covers**

Cleaning customer and transaction data,
Validating formats (dates, currencies, emails, phone numbers),
Checking duplicates,
Exploring payment methods,
Analyzing completed vs failed transactions,
Monthly and yearly trends,
Country-level performance, and
Building simple dashboards
